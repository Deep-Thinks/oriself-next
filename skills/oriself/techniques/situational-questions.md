# Technique · 情境化提问

> Oriself 访谈的第一技术基础。

## 核心规则

**抽象 → 具体场景**。任何"你 X 吗"类问题，都先翻成"最近一次 / 现在 / 那天"开头的具体题。

```
坏  "你外向吗？"
好  "上周一个完整没出门的一天，过到晚上你感觉怎么样？"

坏  "你理性还是感性？"
好  "你最近一次帮朋友做决定，你是怎么想的？"
```

## 单一开放问 vs A/B 选

默认**单一开放问号**，让 TA 自己吐出概念，比给两个让 TA 挑信号强。

但留心 TA 自己的偏好：有人喜欢被给两个甚至更多选项当锚（"对，我就是 A 那种"），有人厌恶被框（"两个都不太是"）。**前 3 轮就把这个观察抓出来**：

- TA 答开放问总是给一个画面 + 自己的判断 → 继续单一开放问
- TA 答开放问只给"嗯 / 还行 / 都可以" → 进入 fallback 触发条件，按 `domains/mbti.md` 标记 `[短回复 fallback]` 的 A/B 题用，**不要自己造 A/B**
- TA 自己反过来给你列 ABC（"我感觉自己介于内向和外向之间，可能更偏 A 一点"）→ TA 偏好结构化思考，可以问"你刚说三档，哪一档最近最常出现？"——但仍是开放追问，不是你硬塞两个名词性选项

判定线：**你列出两个名词性选项 = 违反铁则 1**，无论 TA 偏好如何。fallback 题是题池里固定的、二元真实行为偏好（"早到/踩点"），不是你临时造的"A 还是 B"。

## 禁区 · 抽象问法

- ❌ 你是外向的人吗？
- ❌ 你感性还是理性？
- ❌ 你是 N 人还是 S 人？/ 你的 MBTI 是什么？
- ❌ 你怎么看待自己？
